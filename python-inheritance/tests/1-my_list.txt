The ``1-my_list`` module
========================

Import the class:
    >>> MyList = __import__('1-my_list').MyList

Create an instance:
    >>> my_list = MyList()
    >>> isinstance(my_list, list)
    True

Append integers and print the list:
    >>> my_list.append(1)
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> my_list.append(3)
    >>> my_list.append(5)
    >>> print(my_list)
    [1, 4, 2, 3, 5]

Print the list sorted:
    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]

The original list must not change:
    >>> print(my_list)
    [1, 4, 2, 3, 5]

Empty list:
    >>> my_list = MyList()
    >>> my_list.print_sorted()
    []

Already sorted list:
    >>> my_list = MyList([1, 2, 3, 4, 5])
    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]

Negative integers:
    >>> my_list = MyList([-1, -5, 0, 2, -3])
    >>> my_list.print_sorted()
    [-5, -3, -1, 0, 2]

Repeated integers:
    >>> my_list = MyList([7, 7, 1, 3, 1])
    >>> my_list.print_sorted()
    [1, 1, 3, 7, 7]

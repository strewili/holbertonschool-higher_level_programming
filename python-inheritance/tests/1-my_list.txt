===============================
How to use the MyList class
===============================

First, import the class:
    >>> MyList = __import__('1-my_list').MyList

Create an instance and test print_sorted method:
    >>> my_list = MyList([3, 1, 4, 2])
    >>> print(my_list)
    [3, 1, 4, 2]
    >>> my_list.print_sorted()
    [1, 2, 3, 4]
    >>> print(my_list)
    [3, 1, 4, 2]

Test with an empty list:
    >>> my_list = MyList()
    >>> my_list.print_sorted()
    []

Test with already sorted list:
    >>> my_list = MyList([1, 2, 3, 4, 5])
    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]

Test with negative numbers:
    >>> my_list = MyList([-1, -5, 0, 2, -3])
    >>> my_list.print_sorted()
    [-5, -3, -1, 0, 2]
